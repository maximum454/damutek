Формулировки

Группа «Формулировка вопроса» управляет выбором формулировок вопроса, которые задаются при редактировании каждого задания. Так, автор может указать несколько разных формулировок одного вопроса. Тем самым, у нескольких людей один и тот же вопрос будет звучат по-разному.

Кнопка «Добавить формулировку вопроса» создает дополнительную область для ввода следующего варианта вопроса. Кнопка «Удалить формулировку вопроса» удаляет выбранную область. Учтите, что при удалении формулировки, набранный текст в ее области также будет удален, без сохранения.
