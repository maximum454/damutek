МНОЖЕСТВЕННЫЙ ВАРИАНТ ОТВЕТА

Тестируемому предлагается несколько вариантов ответа, среди которых ему нужно выбрать несколько верных.

Это вид заданий, в которых содержатся несколько правильных ответов, и задача испытуемого состоит в их нахождении.

На странице редактирования этого типа вопроса имеются пять полей для ввода вариантов ответа. Если Вам требуется меньше вариантов ответа, то просто не заполняйте лишние поля — они не будут учитываться автоматически. Текст каждого ответа может быть отформатирован в любое время, в него могут быть внесены изменения.

Среди введенных вариантов нужно отметить несколько верных, т. е. те которые должен будет выбрать тестируемый, чтобы ответ был засчитан верным.
Нельзя ставить отметку верного варианта ответа на пустое поле ответа.
