Что нужно знать
для освоения курса
Для освоения курса предпочтительно знать базовые теоретические сведения о том, для чего
предназначена программа Adobe Photoshop CC.
Для кого этот курс
Курс предназначен для людей, желающих освоить базовые основы работы в Photoshop CC,
развить навык обработки фотографий и изображений, а также их компоновку.
Каких результатов
достигнут студенты
по завершению курса?
Вы сможете самостоятельно пользоваться программой, работать со слоями, деформировать
изображение и текст, накладывать фиьтры и пользоваться эффектами программы, ретушировать
фотографию и готовить файлы для печати.
